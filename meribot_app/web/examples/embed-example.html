MeriBot Widget Embed Example

1. Basic Embed

Add this script to your page to include the MeriBot widget:

<script src="/static/js/meribot-loader.js" data-autoload></script>

2. With Custom Configuration

Customize the widget appearance and behavior:

<script src="/static/js/meribot-loader.js" 
        data-autoload
        data-api-url="http://localhost:3000/chatbot/query"
        data-button-text="Ask MeriBot"
        data-button-color="#3f51b5"
        data-title="MeriBot Assistant"
        data-subtitle="How can I help you today?"
        data-position="bottom-right"
        data-auto-open="false"
        data-debug="false">
</script>

3. Programmatic Initialization

Initialize the widget with JavaScript:

<script src="/static/js/meribot-loader.js"></script>
<script>
    // Initialize with custom config
    const meriBot = initMeriBot({
        apiUrl: 'http://localhost/chatbot/query',
        buttonText: 'Chat Now',
        buttonColor: '#e91e63',
        title: 'MeriBot',
        subtitle: 'How can I assist you today?',
        position: 'bottom-right',
        autoOpen: false,
        debug: false
    });
    
    // You can also control the widget programmatically
    // meriBot.show();
    // meriBot.hide();
    // meriBot.sendMessage('Hello, MeriBot!');
</script>
